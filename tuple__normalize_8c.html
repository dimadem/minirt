void tuple_normalize ( t_tuple *  tuple)

This function converts the given vector into its normalized vector. Normalize is used to convert the vector to a unit vector. Meaning, the length of the given vector will be equal to 1 while still pointing in the same direction.

Warning
This function makes the passed parameter a unit vector. Therefore, it does not return anything.
